<!-- Description -->
NewsSphere is a responsive web app that fetches real-time news using NewsAPI. It includes features like search, category filtering, sorting by date, pagination, and dark mode.

~ Search news articles
~ Filter by category
~ Sort by date
~ Dark mode
~ Pagination
